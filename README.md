# Miniguia de Estudos: Otimização e Administração de Bancos de Dados Relacionais

Contexto e Objetivos
Contexto
No dia a dia do suporte técnico e da administração de infraestrutura, a performance do banco de dados é um pilar crítico. Consultas lentas não apenas prejudicam a experiência do usuário, mas aumentam custos de processamento e podem causar indisponibilidade. Este projeto explora como diagnosticar gargalos e aplicar técnicas de tunning em ecossistemas Oracle e PostgreSQL.

Objetivos de Estudo
Dominar o Diagnóstico: Aprender a ler planos de execução e identificar operações custosas.

Estratégias de Indexação: Diferenciar o uso de índices B-Tree, Bitmap e baseados em função.

Refatoração de Consultas: Aplicar boas práticas de escrita SQL para maximizar a performance.

IA como Mentora: Utilizar Inteligência Artificial para acelerar o troubleshooting de queries complexas.
